# Historical Timeline 3D

Three.jsを用いた、日本史の主要な出来事を3D空間で探索できるインタラクティブなWebアプリケーションです。

![Screenshot](https://via.placeholder.com/800x450.png?text=Historical+Timeline+3D+Preview) <!-- 後で実際のスクリーンショットに差し替え可能です -->

## 概要

このアプリは、時間の流れを「奥行き」として表現し、ユーザーが歴史を遡る体験を提供することを目的としています。教育現場やオープン・スクールでの展示、歴史学習の導入ツールとして活用されることを想定しています。

## 特徴

- **3Dタイムライン**: z軸方向に広がる空間を移動し、時代の移り変わりを体感。
- **インタラクティブ・カード**: 主要な歴史的イベントをクリックして詳細を確認。
- **スムーズな操作感**: GSAPによる滑らかなカメラワークとアニメーション。
- **軽量・高速**: フレームワークを使用せず、シングルHTMLファイルで動作（CDNを利用）。

## 登場する歴史的イベント

1. 鎌倉幕府の成立 (1192)
2. 鉄砲の伝来 (1543)
3. 関ヶ原の戦い (1600)
4. ペリー来航 (1853)
5. 明治維新 (1868)
6. 終戦・平和への道 (1945)
7. 未来を創る現在 (2026)

## 技術スタック

- **3D Engine**: [Three.js](https://threejs.org/)
- **Animation**: [GSAP](https://greensock.com/gsap/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide Icons](https://lucide.dev/)

## 使い方

`history_3d_app.html` をブラウザで開くだけで動作します。

1. **ログイン**:
   - お名前を入力。
   - パスコード **`2026`** を入力して入場。
2. **スクロール**: タイムラインを移動。
3. **クリック**: 各イベントの詳細を表示。
4. **リセットボタン**: 現代の位置へ戻る。（右クリックでログアウト可能です）
